# AI Data Center Atlas

Vercel-ready React + Vite + Leaflet interactive U.S. AI data center map.

## Deploy to Vercel

1. Unzip this folder.
2. Upload all files to your GitHub repository.
3. Make sure the repo structure is:

```text
index.html
package.json
vite.config.js
src/App.jsx
src/main.jsx
src/styles.css
```

4. Go to Vercel → Add New Project → Import your GitHub repo.
5. Click Deploy.

## Local test

```bash
npm install
npm run dev
npm run build
```

## Notes

The map uses OpenStreetMap tiles and estimated wattage values.
